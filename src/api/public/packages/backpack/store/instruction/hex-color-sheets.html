Доступные свойства
  • bold - жирный текст
  • italic - курсивный текст
  • underline - подчеркнутый текст
  • size:{размер_шрифта_числом} - шрифт определенного размера
  • color:{цвет_в_формате_HEX} - текст определенного цвета
  • fill:{цвет_в_формате_HEX} - фон определенного цвета
Примечание
  • Указывайте несколько свойств, чтобы избежать неверной интерпретации данных, чем точнее тем лучше. Например: bold,size:18. То есть - шрифт жирный и его размер: 18
  • HEX-код цвета фона/текста просто так через интерфейс получить нельзя. Однако можно использовать такие варианты:
    • Цветовая пипетка (color picker) через инструменты разработчика в браузере.
    • Цветовая пипетка (color picker) как отдельное приложение для Windows/Mac
    • Загрузить файл в Google Sheets и добавить функцию определения свойств ячейки по интсрукции ниже.
  • Если Категории и Бренды размечены визуально одинаково (полное совпадение) такие данные будут прочитаны неверно.
Как получить свойства ячейки в Google Sheets

В Google Sheets нет встроенной функции, которая напрямую возвращает свойства ячейки. Однако это можно сделать с помощью Google Apps Script в Google Sheets. Эту функция вернет не только цвет фона и шрифта, но и все остальные свойства ячейки. Достаточно будет просто скопировать строки и вставить ее в админку. Вот пошаговая инструкция:

  1. Загрузите файл на Google Disk:
    Загрузите файл (каталог поставщика) в формате xls, xlsx на Google Disk. Откройте файл в Google Sheets (приложение для просмотра excel-таблиц на гугл диске).
  2. Откройте редактор скриптов:
    В Google Sheets нажмите на меню "Расширения" → "Apps Script".
  3. Вставьте код:
    В открывшемся редакторе удалите стандартный код (если он есть) и вставьте следующий скрипт: 
    function getCellProperties(cell) {
  var sheet = SpreadsheetApp.getActiveSheet();
  var range = sheet.getRange(cell);
  
  // Собираем свойства
  var properties = [];
  
  // Жирный текст
  if (range.getFontWeight() === "bold") {
    properties.push("bold");
  }
  
  // Курсив
  if (range.getFontStyle() === "italic") {
    properties.push("italic");
  }
  
  // Подчёркивание
  if (range.getFontLine() === "underline") {
    properties.push("underline");
  }
  
  // Размер шрифта
  var fontSize = range.getFontSize();
  if (fontSize) {
    properties.push("size:" + fontSize);
  }
  
  // Цвет текста (без #)
  var fontColor = range.getFontColor().substring(1);
  properties.push("color:" + fontColor);
  
  // Цвет заливки (без #)
  var background = range.getBackground().substring(1);
  properties.push("fill:" + background);
  
  // Объединяем всё в одну строку через запятую
  return properties.join(",");
}
  4. Сохраните скрипт:
    Нажмите на значок дискеты или выберите "Файл" → "Сохранить", дайте проекту имя, например, "GetCellProperties".
  5. Используйте функцию в таблице:
    Вернитесь в Google Sheets и в любой ячейке введите формулу, например: 
    =getCellProperties("A1")
    Здесь "A1" — это адрес ячейки, свойства ячейки которой вы хотите получить.
  6. Разрешите доступ:
    При первом запуске Google может запросить разрешение на выполнение скрипта. Нажмите "Разрешить", чтобы скрипт мог работать с вашей таблицей.
